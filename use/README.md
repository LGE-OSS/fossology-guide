# How to Use

```note
FOSSology는Open Source Software(OSS) 단위로 Source Code 내의 License Text를 분석합니다.
이에, OSS 파일을 FOSSology에 Upload ([1. OSS 업로드](1_upload.html)) 해야 License 검출 결과를 확인 ([2. Scan 결과 확인](2_scan-result.html)) 할 수 있습니다.

그리고, String Search 에 따라 검출된 License 중 실제로 사용된 License를 확인하는 Clearing([3. Clearing](3_clearing.html)) 과정에 따라 해당 OSS에 포함된 License를 정확하게 확인할 수 있습니다.

마지막으로, FOSSology Scan 결과를 보완하고, Homepage, Download Location 등의 정보를 추가하면 ([OSS List](4_oss-list.html)), 분석 결과를 더욱 효과적으로 공유할 수 있습니다.
```
